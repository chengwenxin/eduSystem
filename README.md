# 部署
1、使用前必须要要有node环境<br>
2、建立数据库<br>
在mysql数据库上新建edusys数据库，然后右键此数据库->运行SQL文件->选择database目录下的文件（编码是utf-8）->点击开始便会生成一个数据库<br>
3、安装依赖<br>
分别在backSys和frontSys目录运行以下代码<br>
npm install<br>
4、运行项目<br>
在backSys目录下运行npm run start<br>
在frontSys目录下运行npm run dev<br>
